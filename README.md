# TCP_demo worked under Ubuntu 22.04
## Project Structure
```bash
tcp_example/
├── include/
│   ├── tcp_common.hpp
│   ├── tcp_server.hpp
│   └── tcp_client.hpp
├── src/
│   ├── tcp_server.cpp
│   ├── tcp_client.cpp
│   └── main.cpp
├── Makefile
└── README.md
```
## Compile & Run
make

## Run server: 
./tcp_example server

##  Run client (separate terminal):
./tcp_example client
